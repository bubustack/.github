# BubuStack GitHub Defaults

This repository stores the organization-level GitHub defaults for BubuStack.

## What lives here

- `profile/README.md` powers the public organization profile at `github.com/bubustack`.
- `CODE_OF_CONDUCT.md`, `CONTRIBUTING.md`, `SECURITY.md`, and `SUPPORT.md` provide default community health files for repositories that do not override them.
- `ISSUE_TEMPLATE/` and `PULL_REQUEST_TEMPLATE.md` define the default issue and pull request flows.
- `labels.yml` and `workflows/label-sync.yml` keep a shared label taxonomy in sync.

## Notes

- The organization profile uses `profile/README.md`, not the root `README.md`.
- Update this repository for org-wide defaults and policies only.
- Repository-specific setup, usage, and architecture docs should live in the affected repository.
